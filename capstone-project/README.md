# Capstone Project

This directory contains materials for the capstone project, the culminating experience of the 12-month program.

## Documents

- **capstone_project_specifications.md** - Complete project requirements, timeline, and deliverables
- **capstone_project_student_resources.md** - Recommended datasets, tools, and research materials

## Overview

In the final quarter (Months 10-12), students work in teams to develop a novel AI-powered solution to a real-world cybersecurity or digital forensics challenge.

## Deliverables

- Project proposal and design document
- Full implementation with well-documented code
- Comprehensive final report (15-20 pages)
- Final presentation and defense

## Suggested Project Areas

- Adversarial attack and defense mechanisms
- AI-powered threat hunting systems
- Deepfake detection and generation
- Automated vulnerability discovery
- AI-driven deception systems (honeypots)

## Timeline

- Month 10: Project selection and proposal
- Month 11: Design and development
- Month 12: Implementation, testing, and presentation

## Grading

The capstone project accounts for 15% of the final grade and is evaluated on:
- Project proposal and design
- Implementation quality
- Final report and documentation
- Presentation and defense
