# Instructor Guides

This directory contains guides for instructors delivering the program.

## Documents

- **general_instructor_guide.md** - General teaching philosophy, classroom management, and best practices
- **lab1_instructor_guide.md** - Specific guidance for teaching Lab 1

## Purpose

These guides provide instructors with:
- Teaching strategies and best practices
- Lab-specific guidance and troubleshooting
- Discussion points and learning objectives
- Grading rubrics and feedback guidelines
- Time management and pacing suggestions

## Using These Guides

1. Read the general instructor guide before starting the program
2. Review lab-specific guides before teaching each lab
3. Use the curriculum documents for week-by-week planning
4. Reference grading rubrics when evaluating student work

## Additional Resources

For additional support, refer to:
- Curriculum documents for detailed content
- Lab assignments for complete specifications
- Assessment framework for grading guidelines
