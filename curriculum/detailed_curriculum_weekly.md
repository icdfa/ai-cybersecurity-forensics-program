# Detailed Weekly Curriculum Breakdown - v4.0

This document provides a week-by-week breakdown of the 24-module fellowship program.

## Phase 1: Core Skills & Defensive Operations (Months 1-6, Weeks 1-26)

| Week | Module | Topics |
| :--- | :--- | :--- |
| 1 | 1: Fellowship Orientation & Professional Development | Program Overview, Tools Setup, GitHub Pro, Professional Communication. |
| 2-3 | 2: Advanced Python for Security Automation | NumPy, Pandas, Scapy, Automation Scripts, Secure Coding Practices. |
| 4-5 | 3: Enterprise Networking & Security (Cisco) | TCP/IP, Routing, Switching, VLANs, ACLs, Cisco Packet Tracer Labs. |
| 6-7 | 4: Cloud Security Engineering (AWS & Azure) | IAM, VPC/VNet, Security Groups, Serverless Security, Cloud Monitoring. |
| 8-9 | 5: Applied Machine Learning for Security | Supervised/Unsupervised Learning, Feature Engineering, Model Evaluation, Azure ML. |
| 10-11 | 6: Deep Learning for Threat Detection | CNNs, RNNs, Transformers, Distributed Training with AWS SageMaker. |
| 12-13 | 7: AI-Powered Threat Intelligence | OSINT Automation, NLP for IoC Extraction, Proactive Threat Hunting. |
| 14-15 | 8: Intelligent Malware Analysis | AI-driven Reverse Engineering, Malware Classification, Behavior Prediction. |
| 16-17 | 9: AI for Advanced Network Defense | Anomaly Detection, Deep Learning NIDS, Encrypted Traffic Analysis. |
| 18-19 | 10: AI-Enhanced Digital Forensics | Automated Artifact Analysis, Large-Scale Memory & Disk Forensics. |
| 20-21 | 11: Natural Language Processing for Security | BERT/GPT for Log Analysis, Phishing Detection, Disinformation Analysis. |
| 22-23 | 12: AI-Driven SOAR & Automation | Intelligent Playbook Orchestration, Reinforcement Learning for IR. |
| 24-26 | **Phase 1 Capstone & Assessments** | Comprehensive practical exam and Phase 1 capstone project. |
