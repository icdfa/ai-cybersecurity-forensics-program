# Contributing to the Advanced AI in Cybersecurity and Digital Forensics Program

We welcome contributions to improve this program. Please follow these guidelines to ensure a smooth and effective collaboration process.

## How to Contribute

1.  **Fork the Repository:** Start by forking the main repository to your own GitHub account.

2.  **Create a Feature Branch:** For any new feature or improvement, create a new branch from the `main` branch.

    ```bash
    git checkout -b feature/your-improvement-name
    ```

3.  **Make Your Changes:** Implement your changes, whether it's new content, bug fixes, or improvements to existing material.

4.  **Commit Your Changes:** Follow the commit message conventions.

    ```bash
    git commit -m "feat: Add new lab on deepfake detection"
    ```

5.  **Push to Your Branch:** Push your changes to your forked repository.

    ```bash
    git push origin feature/your-improvement-name
    ```

6.  **Create a Pull Request:** Open a pull request from your feature branch to the main repository's `main` branch. Provide a clear description of your changes.

## Contribution Guidelines

-   **Content:** Ensure that all new content is well-researched, accurate, and aligns with the program's objectives.
-   **Code:** All code should be well-documented and follow best practices.
-   **Style:** Maintain a consistent writing style and format across all documents.
-   **Issues:** For major changes, please open an issue first to discuss your proposed changes.

## Code of Conduct

All contributors are expected to adhere to the [Code of Conduct](CODE_OF_CONDUCT.md) to ensure a welcoming and inclusive environment.
