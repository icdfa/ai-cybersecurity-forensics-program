# Datasets for AI in Cybersecurity and Digital Forensics

This directory contains recommended datasets for labs and projects throughout the program.

## Malware Datasets

**EMBER** - Elastic Malware Benchmark for Evaluation and Research
- Large-scale dataset of PE file features for malware classification
- Over 1 million samples with labels
- URL: https://github.com/elastic/ember

**Sorel-20M** - Sophos-ReversingLabs Malware Dataset
- 20 million malware samples with rich metadata
- Includes file hashes, tags, and family information
- URL: https://github.com/sophos-ai/Sorel-20M

## Network Traffic Datasets

**CTU-13** - Botnet Traffic Dataset
- Dataset of botnet traffic captured in a real-world network
- Multiple botnet families represented
- URL: https://www.stratosphereips.org/datasets-ctu-13

**CIC-IDS-2017** - Intrusion Detection Dataset
- Comprehensive dataset for intrusion detection
- Contains normal and attack traffic
- URL: https://www.unb.ca/cic/datasets/ids-2017.html

## Digital Forensics Datasets

**Digital Corpora** - Forensic Artifacts Repository
- Repository of disk images, memory dumps, and forensic artifacts
- Useful for practicing forensic analysis
- URL: https://digitalcorpora.org/

**GovDocs** - Government Documents Dataset
- Large dataset of documents for file type identification
- Useful for file carving and recovery exercises
- URL: https://digitalcorpora.org/corpora/govdocs

## Deepfake and Multimedia Datasets

**Deepfake Detection Challenge (DFDC)** - Kaggle Competition Dataset
- Large dataset of deepfake videos
- Useful for training deepfake detection models
- URL: https://www.kaggle.com/c/deepfake-detection-challenge

**FaceForensics++** - Facial Manipulation Dataset
- Dataset of manipulated facial videos
- Includes multiple manipulation techniques
- URL: https://github.com/ondyari/FaceForensics

## How to Use These Datasets

1. Download the dataset from the provided URL
2. Extract and organize the data
3. Follow lab instructions for specific usage
4. Ensure compliance with dataset licenses and terms of use
