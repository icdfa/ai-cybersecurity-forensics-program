# Industry Partnerships

## Strategic Technology Partnerships

The International Cybersecurity and Digital Forensics Academy (ICDFA) has established strategic partnerships with the world's leading technology companies to provide students with enterprise-grade resources and industry-recognized training platforms.

---

## Microsoft for Azure

**Status:** Verified Institution

**Benefits for Students:**
- Free access to Microsoft Azure cloud services for the duration of the program
- Unlimited cloud computing resources for labs and research projects
- Access to Azure AI and Machine Learning services
- Azure Security Center for hands-on security operations
- Azure Sentinel for SIEM and SOAR training
- Azure DevOps for collaborative development

**Program Integration:**
- All 12 labs are designed to leverage Azure cloud infrastructure
- Students deploy AI models, security tools, and forensic platforms on Azure
- Capstone projects can utilize Azure's full suite of services
- Training on enterprise-grade cloud security practices

**Value:** Estimated $500-1,000 per student in cloud credits over 12 months

---

## AWS Academy

**Status:** Member Institution

**Benefits for Students:**
- Official access to Amazon Web Services curriculum
- AWS Educate credits for hands-on practice
- Access to AWS cloud computing resources
- Training on AWS security services (GuardDuty, Security Hub, Inspector)
- Preparation for AWS certification exams

**Program Integration:**
- Labs include AWS deployment options alongside Azure
- Students learn multi-cloud security strategies
- Training on AWS Lambda for serverless security functions
- Integration with AWS threat intelligence services

**Value:** Estimated $300-500 per student in AWS credits and curriculum access

---

## GitHub Campus

**Status:** Approved Institution

**Benefits for Students:**
- Free access to GitHub Pro features
- GitHub Copilot for AI-assisted coding
- Unlimited private repositories
- GitHub Actions for CI/CD pipelines
- GitHub Advanced Security features
- Collaborative development tools

**Program Integration:**
- All lab code is hosted on GitHub
- Students learn version control and collaborative development
- Code review and peer feedback through GitHub
- Portfolio building with public repositories
- Integration with GitHub security scanning tools

**Value:** Estimated $200-300 per student in GitHub services over 12 months

---

## Combined Value Proposition

**Total Value per Student:** $1,000-1,800 in free cloud services, tools, and curriculum access

These partnerships ensure that ICDFA students:
- Train on the same platforms used by Fortune 500 companies
- Gain hands-on experience with enterprise-grade tools
- Build portfolios on industry-standard platforms
- Receive training that directly translates to job readiness
- Access resources that would otherwise be cost-prohibitive
