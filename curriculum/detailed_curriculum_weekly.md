# Detailed Weekly Curriculum Breakdown - v3.0

This document provides a week-by-week breakdown of the 24-module fellowship program.

## Semester 1: Core Foundations & Research Methodology (Months 1-4)

| Week | Module | Topics |
| :--- | :--- | :--- |
| 1 | 1: Fellowship Orientation & Research Methodology | Intro to Fellowship, Research Questions, Literature Review, Paper Structure. |
| 2 | 2: Advanced Python for Scientific Computing | NumPy, SciPy, Pandas, Secure Coding, Performance Optimization. |
| 3-4 | 3: Enterprise Networking & Security (Cisco) | OSI Model, TCP/IP, Routing, Switching, VLANs, ACLs, Cisco Packet Tracer. |
| 5-6 | 4: Cloud Security Engineering (AWS & Azure) | IAM, VPC/VNet, Security Groups, Serverless Security, Cloud Monitoring. |
| 7-8 | 5: Applied Mathematics for Machine Learning | Linear Algebra, Vector Calculus, Probability, Statistics, Gradient Descent. |
| 9-10 | 6: Core Machine Learning & Statistical Analysis | Regression, Classification, Clustering, Dimensionality Reduction, Azure ML. |
| 11-12 | 7: Deep Learning & Neural Network Architectures | CNNs, RNNs, LSTMs, Transformers, Distributed Training with AWS SageMaker. |
| 13 | 8: AI Ethics, Governance & Legal Frameworks | Bias, Fairness, Accountability, NIST AI RMF, EU AI Act, Position Paper. |
| 14-16 | **Semester 1 Exams & Project Presentations** | Written exams on core concepts and presentation of math/ethics research. |

## Semester 2: AI for Defensive Security & Forensics (Months 5-8)

| Week | Module | Topics |
| :--- | :--- | :--- |
| 17-18 | 9: AI-Powered Threat Intelligence | OSINT Automation, NLP for IoC Extraction, Proactive Threat Hunting. |
| 19-20 | 10: Intelligent Malware Analysis | AI-driven Reverse Engineering, Malware Classification, Behavior Prediction. |
| 21-22 | 11: AI for Advanced Network Defense | Anomaly Detection in Encrypted Traffic, Deep Learning NIDS. |
| 23-24 | 12: AI-Enhanced Digital Forensics | Automated Artifact Analysis, Large-Scale Memory & Disk Forensics. |
| 25-26 | 13: Natural Language Processing for Security | BERT/GPT for Log Analysis, Phishing Detection, Disinformation Analysis. |
| 27-28 | 14: AI-Driven SOAR & Automation | Reinforcement Learning for IR, Intelligent Playbook Orchestration. |
| 29 | 15: Research Seminar I: Defensive AI | Weekly presentations and critiques of recent academic papers. |
| 30-32 | 16: Mid-Program Research Project | Group research project in defensive AI, paper and presentation. |

## Semester 3: Offensive AI, Advanced Research & Capstone Thesis (Months 9-12)

| Week | Module | Topics |
| :--- | :--- | :--- |
| 33-34 | 17: Adversarial Machine Learning | Evasion, Poisoning, Model Extraction, Membership Inference Attacks. |
| 35-36 | 18: Red Teaming & Security Validation of AI | AI Red Teaming Methodology, Vulnerability Assessment, Risk Scoring. |
| 37-38 | 19: Generative AI Threats & Deepfake Forensics | LLM Security, Prompt Injection, Advanced Deepfake Detection. |
| 39-40 | 20: AI for Vulnerability Research & Exploit Dev | AI-powered Fuzzing, Symbolic Execution, Assisted Exploit Development. |
| 41 | 21: Research Seminar II: Offensive AI | Weekly presentations on emerging and future offensive AI threats. |
| 42-44 | 22: Capstone Thesis I: Proposal & Initial Research | Finalize thesis proposal, literature review, initial experiments. |
| 45-48 | 23: Capstone Thesis II: Development & Implementation | Intensive development phase with one-on-one mentorship. |
| 49-52 | 24: Capstone Thesis III: Final Paper & Defense | Write final paper, formal defense, and presentation to industry partners. |
