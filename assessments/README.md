## Grading Scale

| Grade | Percentage | Description |
| :--- | :--- | :--- |
| A | 90-100% | Excellent mastery of all concepts and skills |
| B | 80-89% | Good understanding with minor gaps |
| C | 70-79% | Satisfactory understanding with areas for improvement |
| D | 60-69% | Minimal understanding |
| F | Below 60% | Insufficient understanding |

## Assessment Weights

- Lab Assignments: 40%
- Quizzes: 15%
- Quarterly Mini-Projects: 20%
- Midterm Exam: 10%
- Final Capstone Project: 15%
