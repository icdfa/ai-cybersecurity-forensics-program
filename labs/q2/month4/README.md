# Quarter 2, Month 4: Principles of Digital Forensics & AI Integration

## Overview

This month introduces digital forensics principles and explores how AI can enhance forensic investigations. Students learn to use industry-standard tools like Autopsy and Volatility Framework.

## Lab Assignments

### Lab 4: Forensic Analysis of a Compromised System (Weeks 13-14)

Conduct a forensic analysis of a disk image from a compromised Windows system using Autopsy and Volatility Framework.

**Topics Covered:**
- Digital forensics principles and evidence handling
- Disk image analysis with Autopsy
- Memory forensics with Volatility Framework
- Artifact extraction and analysis
- Timeline reconstruction

**Deliverables:**
- Forensic analysis report (3-5 pages)
- Extracted artifacts and evidence
- Timeline of attack events

**Grading:** 100 points

See [lab4_forensic_analysis.md](lab4_forensic_analysis.md) for complete details.

## Learning Outcomes

Upon completion of this month, students will be able to:

- Use Autopsy for detailed disk image analysis
- Use Volatility Framework for memory forensics
- Identify and extract artifacts of malicious activity
- Correlate findings to reconstruct attack timeline
