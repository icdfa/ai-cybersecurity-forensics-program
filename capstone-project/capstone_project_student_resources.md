# Capstone Project: Student Resources

This document provides a list of recommended datasets, tools, and other resources to help you with your capstone project. This is not an exhaustive list, and you are encouraged to explore and use other resources as well.

## 1. Datasets

### Cybersecurity and Malware

-   **EMBER:** A large-scale dataset of PE file features for malware classification. (https://github.com/elastic/ember)
-   **Sorel-20M:** A large dataset of malware samples with rich metadata. (https://github.com/sophos-ai/Sorel-20M)
-   **CTU-13:** A dataset of botnet traffic captured in a real-world network. (https://www.stratosphereips.org/datasets-ctu-13)
-   **CIC-IDS-2017:** A comprehensive dataset for intrusion detection. (https://www.unb.ca/cic/datasets/ids-2017.html)

### Digital Forensics

-   **Digital Corpora:** A repository of disk images, memory dumps, and other forensic artifacts. (https://digitalcorpora.org/)
-   **GovDocs:** A large dataset of documents for file type identification and other forensic tasks. (https://digitalcorpora.org/corpora/govdocs)

### Adversarial AI and Deepfakes

-   **Deepfake Detection Challenge (DFDC):** A large dataset of deepfake videos. (https://www.kaggle.com/c/deepfake-detection-challenge)
-   **FaceForensics++:** A dataset of manipulated facial videos. (https://github.com/ondyari/FaceForensics)

## 2. Tools and Libraries

### Machine Learning and Deep Learning

-   **Scikit-learn:** A comprehensive library for machine learning in Python.
-   **TensorFlow and Keras:** Open-source platforms for deep learning.
-   **PyTorch:** A popular deep learning framework.
-   **ART (Adversarial Robustness Toolbox):** A library for adversarial machine learning.

### Cybersecurity and Digital Forensics

-   **Scapy:** A powerful Python library for network packet manipulation.
-   **Volatility Framework:** A memory forensics framework.
-   **Autopsy:** A digital forensics platform.
-   **Radare2:** A reverse engineering framework.

## 3. Cloud Platforms

-   **Google Colab:** A free Jupyter notebook environment with GPU support.
-   **Kaggle:** A platform for data science competitions with free access to GPUs and datasets.
-   **AWS, Google Cloud, Microsoft Azure:** Cloud platforms that offer powerful computing resources for training large models.

## 4. Research and Papers

-   **arXiv:** A repository of pre-print papers in computer science and other fields.
-   **Google Scholar:** A search engine for academic literature.
-   **Papers with Code:** A resource for finding papers and their corresponding code implementations.
