# Quarter 1: Foundations of AI and Cybersecurity

This quarter introduces students to the fundamentals of AI and machine learning, with a focus on cybersecurity applications.

## Months

- [Month 1: Python for Cybersecurity & Fundamentals of AI/ML](month1/)
- [Month 2: Data for Threat Intelligence & ML for Threat Prediction](month2/)
- [Month 3: Static & Dynamic Malware Analysis with AI](month3/)

## Quarter Objectives

- Understand the basics of Python programming for cybersecurity
- Learn fundamental concepts of artificial intelligence and machine learning
- Apply ML techniques to threat detection and malware analysis
- Build an intelligent malware detection system
