# Quarter 2, Month 6: Multimedia Forensics & AI-Driven Incident Response

## Overview

This month covers multimedia forensics and AI-driven incident response. Students learn to develop SOAR playbooks for automated response to security incidents.

## Lab Assignments

### Lab 6: Developing a SOAR Playbook for Automated Response (Weeks 23-24)

Design and implement a Security Orchestration, Automation, and Response (SOAR) playbook for automated incident response.

**Topics Covered:**
- SOAR platforms and concepts
- Playbook design and workflow
- Observable extraction and enrichment
- Automated remediation actions
- Integration with external tools

**Deliverables:**
- SOAR playbook design document
- Python implementation
- Lab report with analysis

**Grading:** 100 points

See [lab6_soar_playbook.md](lab6_soar_playbook.md) for complete details.

## Learning Outcomes

Upon completion of this month, students will be able to:

- Design workflows for automated incident response
- Integrate external security tools and APIs
- Extract and analyze observables from alerts
- Implement SOAR playbook execution
