# Detailed Weekly Curriculum Breakdown - v2.0

This document provides a week-by-week breakdown of the 12-month fellowship program.

## Quarter 1: Foundations of AI, Data, and Cybersecurity

| Week | Module                                                      | Topics                                                                                                  |
| :--- | :---------------------------------------------------------- | :------------------------------------------------------------------------------------------------------ |
| 1-2  | 1: Advanced Python & Cybersecurity Tooling                  | Advanced Python, Data Structures, OOP, Git, Docker, Virtual Environments, CLI Tools.                    |
| 3-4  | 2: Data Generation for AI-Based Security Systems (Lab 1)    | Network Analysis (Scapy), PCAP Analysis, Packet Labeling, Feature Extraction for ML Models.             |
| 5-6  | 3: Machine Learning Fundamentals for Cybersecurity (New Lab) | Supervised vs. Unsupervised, Classification vs. Regression, Feature Engineering, Model Evaluation.      |
| 7-8  | 4: Threat Intelligence & Data Pipelines (Lab 2)             | OSINT, STIX/TAXII, Building Threat Intel Feeds, SQLite, API Integration, Data Normalization.            |
| 9    | 5: AI Ethics and Governance - Part 1                        | Responsible AI, Bias in AI, Fairness, Accountability, Transparency, Legal Frameworks (GDPR, CCPA).       |
| 10-12| 6: Static & Dynamic Malware Analysis with AI (Lab 3)        | PE File Analysis, String Analysis, Intro to Dynamic Analysis (Sandboxing), Behavioral Feature Extraction. |

## Quarter 2: AI-Powered Digital Forensics & Incident Response

| Week | Module                                                   | Topics                                                                                                |
| :--- | :------------------------------------------------------- | :---------------------------------------------------------------------------------------------------- |
| 13-16| 7: Automated Forensic Analysis of Compromised Systems (Lab 4) | Filesystem Forensics (ext4, NTFS), Memory Forensics (Volatility), AI-driven Artifact Discovery.        |
| 17-20| 8: Automated Attack Timeline Generation (Lab 5)          | Log Parsing (Sysmon, Apache), NLP for Log Analysis, Event Correlation, Building Attack Timelines.       |
| 21-24| 9: AI-Powered SOAR (Lab 6)                               | SOAR Principles, Playbook Development, API Integration, Automating Triage and Response with AI.         |

## Quarter 3: Offensive AI & Adversarial Machine Learning

| Week | Module                                                | Topics                                                                                                |
| :--- | :---------------------------------------------------- | :---------------------------------------------------------------------------------------------------- |
| 25-28| 10: Adversarial Attack Research & Evasion (Labs 7 & 8) | Adversarial ML Theory, Evasion Attacks (FGSM, PGD), Model Robustness Testing, Defenses.               |
| 29-32| 11: Data Poisoning & Backdoor Attacks (Lab 9)          | Data-centric Attacks, Label Flipping, Backdoor Implementation, Defenses (Data Sanitization).          |
| 33-36| 12: Deepfake Generation & Detection (Lab 10)           | Generative Adversarial Networks (GANs), Deepfake Creation, Detection Techniques (Forensic Analysis).    |

## Quarter 4: Advanced AI Security, Governance & Capstone Project

| Week | Module                                                     | Topics                                                                                                |
| :--- | :--------------------------------------------------------- | :---------------------------------------------------------------------------------------------------- |
| 37-40| 13: AI Red Teaming & Security Validation (Labs 11 & 12)    | AI Red Teaming Methodology, Vulnerability Assessment, AI Security Validation, Certification.          |
| 41-42| 14: AI Governance, Risk & Compliance (GRC) - Part 2        | NIST AI RMF, ISO 27001 for AI, Auditing AI Systems, Model Governance, Risk Management.                |
| 43-44| 15: SOC Simulation & Team-Based Incident Response (New Lab) | SIEM Operations, Dashboarding, Ticketing Systems (Jira), Collaborative IR, Reporting.                 |
| 45-52| 16: Capstone Project                                       | Team-based project on a real-world AI security problem. Includes proposal, development, and final demo. |
